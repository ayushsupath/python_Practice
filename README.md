# Python Practice

A collection of Python practice programs covering core logic-building exercises and small beginner projects.

## Contents

### Logic Building
- Armstrong number checker
- Prime number checker
- Reverse an integer

### Mini Projects (with Streamlit UI)
- ATM Simulator
- Password Generator
- Grade Calculator

## How to Run

```bash
pip install streamlit
python <filename>.py
```

For the Streamlit-based projects:
```bash
streamlit run <filename>.py
```
